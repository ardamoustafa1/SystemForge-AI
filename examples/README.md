# SystemForge Showcase Exports

These examples are reviewable, implementation-oriented artifacts that demonstrate the output shape SystemForge AI is designed to produce.

| Example | Focus | Included artifacts |
|---|---|---|
| [SaaS control plane](saas-control-plane/) | Tenant isolation, RBAC, auditability | Architecture brief, Mermaid topology |
| [Marketplace platform](marketplace-platform/) | Idempotent checkout and fulfillment | Docker Compose deployment |
| [AI workflow hub](ai-workflow-hub/) | Queue backpressure and provider resilience | Terraform and Kubernetes |

The examples are intentionally small enough to understand quickly. They are reference exports, not universal production defaults: review networking, secrets, storage, scaling, and cloud-specific requirements before deployment.
